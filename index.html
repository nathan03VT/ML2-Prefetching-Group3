Project Overview

This project presents a novel machine learning approach to predict storage I/O accesses for improved SSD prefetching. Traditional prefetching strategies struggle with irregular access patterns and large address spaces. Our neural network-based solution leverages historical I/O patterns to accurately predict both the offset and size of future storage accesses, achieving up to 51x performance improvements over traditional algorithms.

Implementation Details

Key Features

  • Neural network-based prediction of storage access patterns
  • Dual prediction model: offset and size prediction
  • Regression approach for exact value prediction
  • Up to 51x improvement over traditional prefetchers
  • High prefetch coverage (78.7%) and hit ratio

Technical Architecture

Our system consists of:

  • Data preprocessing and feature extraction pipeline
  • Small neural network model (16x4x1 neurons)
  • Temporal pattern detection using previous I/O accesses
  • Prefetching simulation framework
  • Performance evaluation metrics

Results & Performance

Our approach achieved significant improvements:

  • 6.6× better than Markov Chain (base 1)
  • 15.4× better than Markov Chain (base 2)
  • 51.2× better than Stride algorithm
  • Coverage: 78.7% vs 19% (Markov) and 4.4% (Stride)

Future Work

Planned enhancements include:

  • Deploy model in SSD emulator (FEMU)
  • Test in real system environments
  • Explore ML training workload patterns
  • Implement caching system in emulator
